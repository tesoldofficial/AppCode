# AppCode Prototype

В этой папке теперь один React-макет окна приложения, без лендинга и без набора экранов.

## Запуск

```bash
cd docs/prototypes
npm install
npm run dev
```

По умолчанию Vite поднимет локальный сервер на `http://127.0.0.1:4173/`.

## Структура

- `src/App.jsx` — основной макет окна.
- `src/styles.css` — стили этого экрана.
- `index.html` — entrypoint.
